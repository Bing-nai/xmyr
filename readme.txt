flower.fa文件是我们所使用的拟南芥的最长转录本数据
每个物种对应的文件夹内：
物种名的.tar.gz文件是该物种原始序列数据
treefile文件夹是该物种与拟南芥的建树结果
*_cds.fa文件是物种和拟南芥的cds序列
含有_f或flower的.fa文件是该物种与拟南芥的蛋白序列
test1.txt或group.txt文件是该物种与拟南芥blast最终整理分组后的结果
两个.blast文件是双向blast的两个结果文件
plot文件夹是物种ka/ks计算出的比值作的箱型图，含有两种格式
result文件夹内是计算完成后的文件初步整理的文件
ksks文件夹内是初步计算完成的各文件，未经过任何加工
result_*.txt文件是经过注释等加工的该物种所有ka/ks计算的结果
re_meme文件夹是该物种所有基序
re_meme10文件夹是该物种evalue在e-10以内的基序
所有对应的物种结果文件均通过tags中对应物种名的Downloads-tar.gz下载
